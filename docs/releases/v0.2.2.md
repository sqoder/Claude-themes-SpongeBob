# v0.2.2

Ready-to-paste GitHub Release body for `v0.2.2`.

## Highlights

- Added an open-box path for newer Claude Code installs that ship as a native npm wrapper.
- `init` now installs a managed validated Claude Code runtime when the official install no longer exposes a patchable `cli.js`.
- `remove` restores the original official launcher and removes the managed runtime cache.

## What shipped

- Native-wrapper detection for official `@anthropic-ai/claude-code` installs
- Managed runtime cache under:

```text
~/.claude/hippocode-managed-runtime/
```

- Launcher swap for native `bin/claude.exe` installs
- Safer target discovery when another local `claude` wrapper appears earlier on `PATH`
- Smoke coverage for:
  - managed `init`
  - managed `status`
  - launcher execution
  - managed `remove`

## Install

```bash
npx claude-code-theme-patcher init
```
