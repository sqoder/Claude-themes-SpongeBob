# v0.2.1

Ready-to-paste GitHub Release body for `v0.2.1`.

## Highlights

- Added `init` so users can patch Claude Code and start with a usable theme in one command.
- Added custom theme pack import and sync support.
- Added the npm publish workflow and published `claude-code-theme-patcher@0.2.1`.

## What shipped

- Adapter-based Claude Code patching for validated version `2.1.112`
- Bundled SpongeBob theme switching
- Custom theme pack import via JSON
- One-command onboarding with:

```bash
npx claude-code-theme-patcher init
```

- Custom theme onboarding with:

```bash
npx claude-code-theme-patcher init --theme-pack ./my-themes.json
```

## Install

```bash
npx claude-code-theme-patcher init
```

## Notes

- This release was published to npm manually before the GitHub Release tag was created.
- If you create a GitHub `v0.2.1` Release after the manual npm publish, the publish workflow may fail on duplicate version because `0.2.1` already exists on npm.
- Prefer using this file as the GitHub Release body, then start the automated release flow from `0.2.2`.

## Verification

- `npm run verify:release`
- `npm view claude-code-theme-patcher version`
- `npx --yes claude-code-theme-patcher@0.2.1 --help`
